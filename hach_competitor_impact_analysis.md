# 🎯 Competitor Move-Impact Analysis Framework
## "What If" Analysis Tool for Hach China

---

## 🔍 How to Use This Framework

For each competitor move, analyze:
1. **Impact on Hach** — Revenue, market share, brand perception
2. **Affected segments** — Municipal, Industrial, Lab
3. **Hach response options** — Defend, attack, ignore
4. **Time to react** — Urgent, 3 months, 6 months

---

## 📊 ABB → Impact Analysis

### Scenario 1: ABB Launches New Product

**Example:** ABB launches AeroStar ULTRA (accuracy focus)

| Dimension | Analysis |
|-----------|----------|
| **Impact on Hach** | Neutral-to-positive — ABB raises category visibility |
| **Municipal segment** | 🟡 Risk — ABB positioning accuracy for compliance |
| **Industrial segment** | 🟢 Low risk — Different use case |
| **Hach response** | Emphasize IMS integration + service ecosystem |
| **Timeline** | 3-6 months to counter |

---

### Scenario 2: ABB Acquires Technology Company

**Example:** ABB acquires Real Tech (Jan 2024)

| Dimension | Analysis |
|-----------|----------|
| **Impact on Hach** | 🔴 Negative — ABB gains real-time capability |
| **Municipal segment** | 🔴 HIGH — Real-time is key trend |
| **Industrial segment** | 🟡 Medium — Process water also moves to real-time |
| **Hach response** | Accelerate IMS real-time features, target acquired company's customers |
| **Timeline** | 1-3 months to identify targets |
| **Pipeline impact** | Identify 10-20 accounts using Real Tech → Sales priority |

---

### Scenario 3: ABB Lowers Prices

## 💡 Key Insight

The most actionable framework is **"Competitor Move → Account Impact → Pipeline Action"**

Instead of:
> "ABB is launching a new product"

Do this:
> "ABB launched X product. It affects customers using Y. We should target 15 accounts currently on ABB. Action: Sales to prioritize demo requests."
