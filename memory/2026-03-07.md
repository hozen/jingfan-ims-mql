## Key Design Decisions

1. **a-tier-demo.html as primary shareable page** - User explicitly preferred over hach_top10_mvp.html due to richer app-like UI

2. **Stakeholder map as separate file** - Used React + Babel CDN approach (~1MB) rather than converting to vanilla JS
   - Opens in new tab from Profile button
   - Account name passed via URL param (?account=xxx)
   - Can be refined independently with Claude

3. **🔒 Internal intel ≠ fake data** - Personnel changes, contract dates, sales feedback are inherently internal CRM data - honestly labeled rather than pretending to be demo data

4. **Citation links required** - Every data point must have verifiable source URL for human team verification
