Step 1 — 一次性建档（每个A类客户）
  • 执行者：市场团队（AI完成80%，销售验证20%）
  • 输出：账户情报卡（公司概况 + 决策人图谱 + 合规驱动 + 产品空白 + 竞争态势）
  • AI工作量：企查查 + 政府采购网 + 行业媒体 → 自动生成结构化档案
  • 人工工作量：销售代表审阅、补充关系层面信息（约1小时/客户）
  • 频率：建立后持续更新，非重复动作
Step 2 — 每周信号监控（全自动）
  • 执行者：AI全自动，仅高优信号触达人工
  • 监控范围：招标公告 / 人事变动 / 竞品动态 / 政策更新 / 企业新闻
  • 输出：每周信号摘要推送至企业微信，红色信号24小时内推送
  • 人工工作量：每周 <30分钟（仅查看推送，无需主动搜索）
  • 工具：企查查API + 政府采购网RSS + 行业媒体 + AI摘要
Step 3 — 月度账户脉冲报告
  • 执行者：AI生成草稿，市场经理快速审核
  • 输出：账户健康状态（绿/黄/红）+ 本月关键变化 + 建议动作
  • 覆盖：全部50个A类账户，集中在每月1日生成
  • 人工工作量：每月1-2小时（审核标红账户，其余快速浏览）
  • 分发：相关销售代表 + 市场主管 + 销售经理
Step 4 — 拜访前简报（自动触发）
  • 触发：销售在CRM中创建/确认拜访记录时自动生成
  • 输出：一页中文简报（联系人画像 + 账户现状 + 建议议程 + 话术 + 注意事项）
  • 交付：拜访前24小时推送至销售代表企业微信
  • 人工工作量：销售代表阅读 5-10 分钟，零额外准备
Step 5 — 拜访执行（销售负责）
  • 执行者：销售代表（市场提供素材支持）
  • 工具：简报 + 产品资料包 + 竞品对比卡（市场预制）
  • 要求：拜访后24小时内完成情报录入（见Step 6）
  • AI支持：若拜访过程有录音，可自动转录并提取关键信息
Step 6 — 拜访后情报录入（AI辅助）
  • 执行者：销售代表录入笔记，AI提取结构化情报
  • AI工作：从原始笔记中提取竞品线索 / 采购信号 / 关系健康度 / CRM更新建议
  • 输出：结构化情报 → 自动更新账户卡 + 反馈至策略团队
  • 人工工作量：销售代表5分钟填写表单，AI完成其余处理
Step 7 — 季度深度复盘
  • 执行者：销售经理 + 市场经理共同主导，AI提供数据底稿
  • 输出：季度账户SWOT + 下季机会优先级 + 下季拜访计划 + 市场支持需求
  • 覆盖：全部50个A类账户，每季度一次
  • 人工工作量：每账户30分钟（AI完成数据汇总，人工做判断）
